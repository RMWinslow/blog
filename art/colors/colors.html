The Universal Color Language

The ISCC–NBS System of Color Designation is an attempt to build a dictionary of surface colors by combining basic color names with adjectives. It was described by the National Bureau of Standards in the 1976 publication Color: Universal Language and Dictionary of Names The system is meant to be perceptually uniform and is based on the Munsell color system, which describes colors in terms of hue, chroma, and value. Below, I've reproduced some tables of hexidecimal color codes from various attempts to convert this system to sRGB.

Paul Centore's sRGB Conversion (2016)

Paul Centore constructed a set of representative RGB values for each of the colors in the ISCC-NBS system. The details are here, the text file of color codes is here, and a set of data files can be found here.

Converting the values from an old book. (2003)

Here's an older attempt to convert the NBS system to a list of RGB codes. This one takes the list of Munsell values for each color-name from the NBS source, and then converts those Munsell values to RGB directly. However, that's easier said than done.

  • David Mundie was the first to try doing this, but his list has several transcription errors and his 90s-era Mac didn't render RGB correctly.
  • John Foster corrected some of these transcription arrors and then reconverted the Munsell values to RGB. Whenever the Munsell values fell outside of the RGB gamut, he tweaked them to fit them inside. But there are still some issuses with the list.
